    Root Cause Localization Method of Microservice Architecture
                          Based on Knowledge Graph

                            LIU Jin, WANG Yue, LI Zhi-gang, DONG Zhi-qiang, GUO Hao

                                        ( CNOOC Information Technology Center, Beijing 100010, China )

Abstract: In order to improve the effect of microservice architecture fault location, a method of microservice architecture fault root cause lo-
             cation based on knowledge map technology is proposed. Based on multidimensional association analysis algorithm, a multidimen-
             sional data classification model for microservice systems is constructed to determine whether a failure occurs. It builds a connec-
             tion topology diagram of the microservice architecture, uses the PageRank algorithm of the knowledge map to dynamically adjust
             the weight relationship of each service component in the topology diagram, calculates the service component with the highest
             probability of failure as the root cause of the failure, and achieves the root cause location of the microservice architecture. The
             test results show that the accuracy rate of the proposed method's positioning results is higher than 0.75, the recall rate is above
             0.71, and the F1 score is above 0.75. This effectively improves the fault location effect of the microservice architecture, with
             strong practicality and reliability.

Keywords: knowledge graph; microservice architecture; root-cause localization; PageRank algorithm
